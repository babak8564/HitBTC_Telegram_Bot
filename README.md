# HitBTC_Telegram_Bot
*Getting Volume and Price for any pair and checking withdrawal of any coin in HitBTC exchange.*

> Use the following command in order to communicate with bot.
> 
> **command**
> 
>> - /is_withdraw_enable - coin
>> 
>> - /volume - pair
>> 
>> - /price - pair
>>
>> For Example:
>> 
>> /is_withdraw_enable - doge
>> 
>> /volume - ethbtc
>> 
>> /price - ltcusdt
>
>
